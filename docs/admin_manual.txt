Руководство администратора RentBrigadir

Панель управления заявками

Доступ к админ-панели
1. Откройте бота в Telegram
2. Отправьте команду /admin
3. Если вы добавлены в список ADMIN_IDS - откроется панель управления

Основные функции админ-панели:

 Просмотр заявок
- "Новые заявки" - список необработанных заявок
- "Все заявки" - полная история заявок (последние 15)
- "Обновить" - актуальный список новых заявок

Действия с заявками

Для каждой заявки доступны:

1. "Обработано" - отмечает заявку как выполненную
   - Заявка перемещается в архив
   - Клиент получает уведомление
   - Статистика обновляется

2. "Позвонить" - показывает телефон клиента
   - Открывает всплывающее окно с номером
   - Можно скопировать для звонка

3. Детальный просмотр - нажмите на заявку в списке для подробной информации

Статистика

"Статистика" - общая аналитика по заявкам:
  - Всего заявок
  - Новые/обработанные
  - Уникальные клиенты
  - Эффективность обработки

Управление каталогом инструментов

Обновление цен и ассортимента
1. Отредактируйте файл `tools.csv`
2. Перезапустите бота
3. Данные автоматически импортируются
